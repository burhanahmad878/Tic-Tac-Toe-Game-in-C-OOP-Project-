**Description**
This is a console-based Tic Tac Toe game developed in C++ using Object-Oriented Programming (OOP).
Two players can play on the same system. The game validates moves, detects winners, and announces a draw when the board is full.

**Features**
Implemented using C++ classes and OOP principles.
Player vs Player gameplay.
Detects winning conditions (rows, columns, diagonals).
Handles draw scenarios.
Input validation to prevent invalid moves.
Clean console-based interface for easy gameplay.

**How to Play**
Run the program.
Players take turns to enter a number (1-9) to place their mark (X or O).
The game automatically updates the board after each move.
The winner is announced when a player gets 3 in a row, column, or diagonal.
If all spots are filled without a winner, the game ends in a draw.

**Technologies Used**
C++ Programming Language
Object-Oriented Programming (OOP) Concepts
